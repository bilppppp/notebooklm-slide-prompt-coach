# 安装 notebooklm-slide-prompt-coach

如果你的环境支持 `npx skills add`，直接执行：

```bash
npx skills add https://github.com/bilppppp/notebooklm-slide-prompt-coach
```

如果你更习惯指定仓库路径，也可以：

```bash
npx skills add https://github.com/bilppppp/notebooklm-slide-prompt-coach/tree/main
```

安装后，这个 skill 适合在下面这类任务里使用：

- 把参考图翻成 NotebookLM 的 slide 提示词
- 让 NotebookLM 幻灯片更接近某种风格
- 直接产出可粘贴的 YAML 提示词

你可以这样对 AI agent 说：

```text
请用 notebooklm-slide-prompt-coach，帮我根据参考图和材料，直接生成可贴到 NotebookLM 的 YAML 幻灯片提示词。
```
